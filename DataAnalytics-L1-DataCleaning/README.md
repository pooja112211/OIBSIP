
# Retail Sales Data Cleaning

##  Objective

The objective of this project is to clean and preprocess the retail sales dataset to improve data quality and prepare it for accurate analysis.

---

##  Dataset

This project uses the same Retail Sales Dataset used in **Task 1 (Retail Sales Data Analysis)**.

---

##  Technologies Used

- Python
- Pandas
- NumPy
- Jupyter Notebook

---

##  Data Cleaning Steps

The following preprocessing steps were performed:

- Checked dataset structure
- Checked data types
- Identified missing values
- Removed duplicate records
- Corrected inconsistent categorical values
- Standardized Gender column values
- Verified numerical columns
- Checked for null values after cleaning
- Prepared the dataset for analysis

##  How to Run

1. Clone the repository.
2. Install required libraries.
3. Open the notebook.
4. Run all cells.

---

##  Result

The dataset was successfully cleaned and standardized, making it suitable for Exploratory Data Analysis and further machine learning or business intelligence applications.
