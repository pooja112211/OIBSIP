
# Retail Sales Data Analysis

##  Objective

The objective of this project is to analyze retail sales data using Python and perform Exploratory Data Analysis (EDA) to discover meaningful business insights through visualizations.

---

##  Dataset

Retail Sales Dataset

The dataset contains information such as:

- Order ID
- Customer ID
- Gender
- Age
- Date
- Status
- Sales Channel
- Product Category
- Quantity
- Amount
- State
- City

---

##  Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

##  Analysis Performed

- Data Inspection
- Data Cleaning
- Monthly Sales Analysis
- Quarterly Sales Analysis
- Gender-wise Sales
- Age Group Distribution
- Category-wise Revenue
- Sales Channel Analysis
- Top Selling Products
- State-wise Sales
- Order Status Analysis

---

##  Key Insights

- March recorded the highest sales among all months.
- Women contributed more to overall sales than men.
- Customers aged 26–35 placed the highest number of orders.
- Amazon was the highest-performing sales channel.
- "Set" generated the highest revenue among product categories.
- Maharashtra was among the top contributing states in sales.
