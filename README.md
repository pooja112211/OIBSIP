# OIBSIP
This repository contains the projects completed as part of the **Oasis Infobyte Data Analytics Internship (OIBSIP)**.

##  Completed Tasks

###  Task 1 - Retail Sales Data Analysis
**Folder:** `DataAnalytics-L1-EDARetailSales`

Performed Exploratory Data Analysis (EDA) on a retail sales dataset to identify sales trends, customer behavior, top-performing categories, and business insights through data visualization.

###  Task 3 - Data Cleaning
**Folder:** `DataAnalytics-L1-DataCleaning`

Performed data preprocessing and cleaning on the retail sales dataset by handling missing values, removing duplicates, correcting inconsistent values, and preparing the dataset for analysis.
